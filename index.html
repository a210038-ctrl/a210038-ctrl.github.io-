Laboratory Activities

Lab 1 – UI Development

Introduction to Jetpack Compose and Android UI design fundamentals.

Reflection:

In this lab, I learned the fundamentals of Jetpack Compose and Android UI development. I explored how to build layouts using composable functions and organize interface components effectively. This lab provided the foundation for all subsequent mobile application development tasks.

Lab 2 – Interaction

Learning state management and interactive user interfaces.

Reflection:

This lab introduced state management and user interaction handling in Compose. I learned how applications respond dynamically to user actions such as button clicks and input fields. The experience improved my understanding of reactive UI development.

Lab 3 – Material Design

Applying Material Design 3 principles and components.

Reflection:

I learned how to apply Material Design principles to create modern and visually appealing user interfaces. By using Material 3 components, I was able to improve consistency, accessibility, and overall user experience within my application.

Lab 4 – Navigation

Implementing multi-screen navigation using Navigation Compose.

Reflection:

This lab focused on implementing navigation between multiple screens using Navigation Compose. I learned how to pass data between screens and manage navigation routes effectively. This knowledge became essential when developing larger applications.

Lab 5 – Room Database

Using Room Database for local data persistence.

Reflection:

In this lab, I learned how to use Room Database for local data persistence. I implemented database entities, DAO operations, and repositories to store and retrieve user information. This enabled offline data access within the application.

Project 2 – StuQuest

Problem Statement:

Students often require additional motivation, progress tracking, and interactive learning tools to remain engaged in their studies. Project 2 enhances StuQuest by integrating local storage, cloud services, web APIs, and mobile hardware sensors to create a smarter and more engaging educational platform.

Lessons Learned:

Project 2 significantly expanded my technical knowledge in Android development. I successfully implemented Room Database for local persistence, Firebase Firestore for cloud integration, Retrofit API for fetching motivational quotes from the internet, and Camera Sensor integration for capturing study evidence.
